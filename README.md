# 🚀 Universal Video Spider (通用视频爬虫)

Universal Video Spider 是一个具备高级反反爬机制、支持多流态（传统 TS 与现代 fMP4）解析、自带图形用户界面（GUI）与任务队列的自动化视频下载神器。

本项目历经数次核心重构，旨在解决复杂目标网站的 CDN 限流封禁、虚假广告流混淆以及 FFmpeg 容器合并崩溃等业界常见痛点。

---

## ✨ 核心特性

* **智能嗅探与决策**：内置 Playwright 无头浏览器引擎，穿透复杂网页逻辑，自动提取 M3U8 / MP4 真实媒体流。
* **硬核广告剥离 (深度探测法)**：拒绝盲猜！通过并发探针深度解析所有候选流的切片总数，依靠数学逻辑精准锁定拥有数百切片的正片流，彻底抛弃几秒钟的伪装广告。
* **全格式容器兼容**：不仅支持传统的 `.ts` 列表切片，更原生支持最新的 **fMP4 (Fragmented MP4)** 格式。自动分离并下载 `EXT-X-MAP` 初始化文件 (`init.mp4`)，并采用底层二进制流无缝咬合拼接，彻底告别 FFmpeg 的轨道报错。
* **自动画质跃升**：识别到多画质变体列表 (Variant Playlists) 时，自动对比带宽参数，强制抓取 1080p/4K 最高清视频源。
* **动态并发与双模式引擎**：
    * 🔴 **高速爆发模式**：解除封印，30 线程火力全开，适合大厂无限制直连。
    * 🟠 **低速稳定模式**：启用指数退避重试 (Exponential Backoff) 与随机延迟，防尖峰、防封禁，从容应对小站的严苛 503 限流。

## 🔄 更新日志 (Changelog)

### [v1.4] - 终极进化：智能切片探针防广告
* **Fixed**: 修复了目标网站使用广告流混淆视听，导致最终只下载到 6 秒钟广告视频的致命问题。
* **Added**: 重写 `_sniff_real_url` 决策大脑，新增 `_select_best_m3u8` 深度探测机制。现在当嗅探到多个 M3U8 时，会在后台并发试探所有候选者，解析切片数量，强制选中“切片数量最多”的真流，从逻辑上彻底终结下载到广告的可能。
* **Improved**: 进一步扩充了基础 URL 广告词库黑名单。

### [v1.3] - 攻克 fMP4 新型格式与画质降级
* **Fixed**: 修复了由于目标网站采用 fMP4 格式导致 FFmpeg 抛出 `could not find corresponding track id` 和 `Invalid data found` 的合并崩溃问题。
* **Added**: 新增对 fMP4 的 `init.mp4` (EXT-X-MAP) 头文件的单独下载支持，并在合并阶段放弃传统 concat 列表，改用 Python 底层二进制模式 `Init + Seg1 + Seg2...` 强行缝合，最后由 FFmpeg 修复容器时间戳。
* **Improved**: 修改 M3U8 解析策略，自动读取 `Bandwidth` 带宽参数，不再抓取默认的 240p，强制锁定最高画质流。

### [v1.2] - 引入“双流派”下载模式 UI
* **Added**: 在图形界面 (GUI) 增加一键切换按钮，支持在【高速爆发模式】与【低速稳定模式】之间自由切换。
* **Improved**: 任务队列现已支持状态记忆。每个任务在添加到队列时会自动绑定当前的模式参数，实现爬虫内核对队列任务资源的动态、独立调配。

### [v1.1] - 修复 503 限流报错与合并崩溃
* **Fixed**: 修复了遇上防爬虫机制触发 `503 Service Temporarily Unavailable` 导致切片彻底失败的问题。
* **Added**: 引入指数退避重试机制，遇到 503 限流时自动递增休眠时间 (2s, 4s, 6s...)，为 CDN 留出缓冲余地。
* **Added**: 引入微小随机延迟 `random.uniform(0.1, 0.6)` 打散并发尖峰。
* **Improved**: 增加合并容错机制。在交给 FFmpeg 合并前，自动校验并剔除彻底下载失败的废弃切片路径，防止 FFmpeg 因 `No such file` 报错而中断整个输出流程。

## 🛠️ 依赖环境

请确保安装了以下 Python 库及系统组件：
* **Python 3.8+**
* `requests`
* `aiohttp`
* `m3u8`
* `cryptography` (用于 AES 解密)
* `playwright` (需执行 `playwright install chromium` 安装无头浏览器)
* **FFmpeg** (必须安装配置好系统环境变量)
