# 🌟 FireflyTools (流萤媒体工具箱)

![Python Version](https://img.shields.io/badge/Python-3.8%2B-blue.svg)
![PyQt6](https://img.shields.io/badge/PyQt-v6-green.svg)
![Playwright](https://img.shields.io/badge/Playwright-Supported-orange.svg)

**FireflyTools (流萤媒体工具箱)** 是一款基于 PyQt6 打造的现代化、沉浸式桌面工具集。本项目拥有无边框圆角视窗与根据壁纸动态提取主色调的磨砂玻璃 UI，并在优雅的外表下封装了四个强大的自动化媒体处理模块。

无论你是需要从复杂的网页中强行嗅探并下载受保护的视频流，还是需要对本地海量的媒体文件进行批量提取、归档与智能裁剪，流萤工具箱都能为你提供多线程、防阻塞的丝滑体验。

---

## 🎨 全局特性 (Global Features)

* **沉浸式动态 UI**：无边框圆角窗口设计，支持一键切换全局壁纸。
* **智能色彩引擎**：底层自动提取当前壁纸的主色调，动态生成对应的 QSS 样式表，实现按钮、输入框、导航栏色彩的自适应融合。
* **多线程与异步调度**：所有高耗时任务（如 Playwright 爬虫、FFmpeg 合并、大批量文件 I/O）均在后台线程/异步协程中运行，彻底告别界面卡死。

---

## 📦 核心功能模块 (Core Modules)

### 🎬 1. Universal Video Spider (通用视频爬虫)

> 本项目最核心的杀手锏模块。一个具备高级反反爬机制、支持多流态（传统 TS 与现代 fMP4）解析、自带任务队列的自动化视频下载神器。旨在解决复杂目标网站的 CDN 限流封禁、虚假广告流混淆以及 FFmpeg 容器合并崩溃等业界常见痛点。

**✨ 爬虫核心特性：**

* **智能嗅探与决策**：内置 Playwright 无头浏览器引擎，穿透复杂网页逻辑，自动提取 M3U8 / MP4 真实媒体流。
* **硬核广告剥离 (深度探测法)**：拒绝盲猜！通过并发探针深度解析所有候选流的切片总数，依靠数学逻辑精准锁定拥有数百切片的正片流，彻底抛弃几秒钟的伪装广告。
* **全格式容器兼容**：不仅支持传统的 `.ts` 列表切片，更原生支持最新的 **fMP4 (Fragmented MP4)** 格式。自动分离并下载 `EXT-X-MAP` 初始化文件 (`init.mp4`)，并采用底层二进制流无缝咬合拼接，彻底告别 FFmpeg 的轨道报错。
* **自动画质跃升**：识别到多画质变体列表 (Variant Playlists) 时，自动对比带宽参数，强制抓取 1080p/4K 最高清视频源。
* **动态并发与双模式引擎** (支持 UI 实时切换)：
  * ⚡ **高速爆发模式**：解除封印，30 线程火力全开，适合大厂无限制直连。
  * 🛡️ **低速稳定模式**：启用指数退避重试 (Exponential Backoff) 与随机延迟，防尖峰、防封禁，从容应对小站的严苛 503 限流。

**🔄 爬虫模块更新日志：**

* **[v1.4] 终极进化：智能切片探针防广告**
  * *Fixed*: 修复了目标网站使用广告流混淆视听，导致最终只下载到 6 秒钟广告视频的致命问题。
  * *Added*: 重写 `_sniff_real_url` 决策大脑，新增 `_select_best_m3u8` 深度探测机制。现在当嗅探到多个 M3U8 时，会在后台并发试探所有候选者，解析切片数量，强制选中“切片数量最多”的真流，从逻辑上彻底终结下载到广告的可能。
* **[v1.3] 攻克 fMP4 新型格式与画质降级**
  * *Fixed*: 修复了由于目标网站采用 fMP4 格式导致 FFmpeg 抛出 `could not find corresponding track id` 等合并崩溃问题。
  * *Added*: 新增对 fMP4 的 `init.mp4` (EXT-X-MAP) 头文件的单独下载支持，采用 Python 底层二进制模式强行缝合，最后由 FFmpeg 修复容器时间戳。
* **[v1.2] 引入“双流派”下载模式 UI**
  * *Added*: 增加一键切换按钮，支持在【高速爆发模式】与【低速稳定模式】之间自由切换，且队列任务支持独立模式记忆。

### 📂 2. Video Extractor (视频子目录提取)

一键扫描指定主文件夹及其所有子目录，将深埋在复杂层级中的视频文件（mp4, mkv, avi 等）全部提取到根目录。

* 自带智能防覆盖重命名逻辑 (`_1`, `_2`)。
* 可选择在提取完成后，自动清理残留的空文件夹。

### 🗂️ 3. Keyword Organizer (关键字归档)

告别杂乱无章的媒体库。输入指定的关键字，工具会在后台静默扫描当前目录，将所有文件名包含该关键字的图片与视频自动移动到对应的专属归档文件夹中。

* 严格的文件格式过滤，防止误伤系统文件。
* 智能懒加载：仅在匹配成功时创建文件夹，杜绝空目录泛滥。

### 🖼️ 4. Smart Image Resizer (图片智能裁剪)

提供图片批处理缩放与裁剪功能，底层调用强大的 PIL (Pillow) 库，采用 LANCZOS 高质量重采样算法。

* 支持自定义输出分辨率，处理带透明通道的图层，自动转为白底 RGB。
* 支持智能重心偏移（居中/保头/保脚），满足不同长宽比的裁剪需求。

## 🛠️ 依赖环境与安装指导

在运行本工具箱之前，请确保你的系统满足以下条件：

1. **安装 Python 3.8+**

2. **安装核心第三方库**：
   打开终端或命令行，运行以下命令：

   ```bash
   pip install PyQt6 Pillow requests aiohttp m3u8 cryptography playwright
   ```

3. **安装 Playwright 浏览器内核**（用于视频嗅探）：

   Bash

   ```
   playwright install chromium
   ```

4. **配置 FFmpeg**：

   本工具的视频合并功能高度依赖 `FFmpeg`。请前往 [FFmpeg 官网](https://ffmpeg.org/download.html) 下载对应你操作系统的版本，解压后务必将其 `bin` 目录添加到系统的 **环境变量 (PATH)** 中。

------

## 🚀 快速启动

1. 将本仓库克隆到本地。

2. 确保在工具箱根目录下包含一个名为 `pic` 的文件夹，并在里面放入几张 `.jpg` 或 `.png` 壁纸，用于启动动态 UI 引擎。

3. 运行主程序：

   Bash

   ```
   python main.py
   ```
